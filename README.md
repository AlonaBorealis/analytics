# analytics

## Setup

```
npm install
```

## Scripts

- `npm run dev` — start dev server with live-reload on http://localhost:3000
- `npm run build` — compile `scss/main.scss` to `css/style.css`
- `npm run watch` — compile scss on change
- `npm run lint` — lint JS with ESLint
- `npm run format` — format the project with Prettier
- `npm run format:check` — check formatting without writing changes
